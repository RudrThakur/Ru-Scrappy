# Ru-Scrappy
NodeJS | Cheerio | Web Scraper

## Installation

```bash

npm install
npm run start

```

## Example

```javascript 
    path-to-server/api/scrape

    // Request Type = POST
    // Request Body 
    {
        "url": "your-preffered-url"
    }
```

## Endpoints

```javascript

// use cheerio
path-to-server/api/scrape

// version 2
path-to-server/api/scrape/v2

// version 3
path-to-server/api/scrape/v3

// use unfluff
path-to-server/api/scrape/unfluff

// use jsdom
path-to-server/api/scrape/jsdom

```
