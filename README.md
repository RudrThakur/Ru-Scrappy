# Ru-Scrappy
NodeJS | Cheerio | Web Scraper

## Installation

```bash
npm install express
npm install cheerio
npm install got
```

## Example

API Endpoint - 

```javascript 
    path-to-server/api/scrape

    // Request Type = POST
    // Request Body 
    {
        "url": "your-preffered-url"
    }
```
